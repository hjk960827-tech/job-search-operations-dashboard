시작 순서

  1. npm run setup으로 로컬 설정을 만듭니다.
  2. APP_MODE=personal npm run db:init으로 빈 DB를 만듭니다.
  3. APP_MODE=personal npm run dashboard로 개인 화면을 엽니다.

로컬 설정과 DB는 Git에 포함되지 않습니다.
