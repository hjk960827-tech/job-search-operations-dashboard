# GitHub Codespaces 체험

Codespaces를 사용하면 개인 컴퓨터에 Node.js나 이 저장소를 설치하지 않고 실제 대시보드 화면을 브라우저에서 열 수 있습니다.

1. README의 `Open in GitHub Codespaces` 버튼을 누릅니다.
2. GitHub의 `Create codespace`를 누르고 설치가 끝날 때까지 기다립니다.
3. 대시보드가 자동으로 시작되면 `Open in Browser` 알림을 누릅니다. 알림을 놓쳤다면 `PORTS`에서 `8766`의 지구본 아이콘을 누릅니다.
4. 첫 실행에는 11단계 초기 설정 화면이 열립니다. 기존 설정을 완료한 Codespace를 다시 열면 개인 모드로 시작합니다.

## 개인정보 주의

- 전달 포트는 기본값인 `Private`로 유지하고 `Public`으로 변경하지 마세요.
- Codespaces는 내 컴퓨터가 아니라 GitHub의 원격 환경입니다. 기능 확인에는 합성 이력서와 합성 포트폴리오만 사용하세요.
- 실제 이력서, 연락처, 지원 이력과 Codex·Claude 로그인 정보는 개인 컴퓨터에 설치한 환경에서만 사용하세요.
- 체험을 마치면 `https://github.com/codespaces`에서 Codespace를 중지하거나 삭제할 수 있습니다.

문서 분석 단계는 대시보드가 AI를 내장 호출하지 않으므로 구조화 분석 JSON을 직접 붙여넣거나 별도로 로그인한 Codex·Claude 작업이 필요합니다.
